# PRGate

**Policy-aware code review for pull requests**

PRGate is an open-source pull request review platform that combines policy, context, and AI.

## What PRGate Is

PRGate is being built as an organization-wide review layer for engineering teams that want code review workflows to be more consistent, more context-aware, and easier to govern across repositories.

The public project direction is centered on:

- policy-aware pull request review
- GitHub-native workflows
- provider-agnostic AI integrations
- clear review outcomes that reduce noise instead of adding more of it

## Repository Layout

The initial public repository layout is:

- `.github` for the org profile and shared community-health files
- `prgate` for the core product repository
- `prgate-examples` for examples and sample integrations

## Getting Started

- Use this `.github` repository for shared issue templates, pull request guidance, and organization defaults.
- As public repositories are published, `prgate` will be the main home for product code and docs.
- `prgate-examples` will hold sample setups and demo integrations.

## Current Status

This organization is being set up in public. The goal of this repository is simple: explain what PRGate is, make contribution paths clear, and provide shared defaults for the repositories that follow.
