# PRGate `.github`

This repository holds the PRGate organization profile and shared community-health files.

PRGate is an open-source pull request review platform that combines policy, context, and AI.

## What Lives Here

- the GitHub organization profile in `profile/README.md`
- shared issue templates and the default pull request template
- contributor-facing guidance such as `CONTRIBUTING.md`, `SECURITY.md`, and `SUPPORT.md`

## Repository Map

The org is organized around a small set of repositories:

- `.github` for organization profile and shared GitHub defaults
- `prgate` for the main product codebase and public documentation
- `prgate-examples` for sample configurations and demo integrations

This repository is intentionally light on product internals. Public implementation details should live in the relevant product repositories as they are published.
